# obrabotka izobrazheniy

XYZ - это цветовое пространство, созданное чтобы математически описывать, как человек воспринимает цвет, используется для точной цветопередачи, измерений, перевода между моделями.
RGB - это цветовая модель для экранов, зависит от устройства, используется для отображения цветов на экране.
